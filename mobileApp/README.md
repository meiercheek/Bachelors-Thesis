# Mushroom App

Application that serves as a guide for classifiying mushrooms.

Supported classes of mushrooms:
```
Agaricus arvensis
Amanita muscaria
Amanita pantherina
Amanita phalloides
Auricularia auricula-judae
Boletus reticulatus
Calocera viscosa
Cantharellus cibarius
Chlorociboria aeruginascens
Cortinarius semisanguineus
Cortinarius violaceus
Galerina marginata
Gyromitra esculenta
Hygrocybe cantharellus
Lactarius deliciosus
Leccinum scabrum
Lepista nuda
Macrolepiota procera
Pleurotus ostreatus
Rubroboletus satanas
Russula cyanoxantha
Russula lepida
Sarcoscypha austriaca
Suillus luteus
```

Ways to launch:

* Install `.apk` file on your android device, remember to enable 'Installation from untrusted sources'

or:

* Open this folder in Android Studio, build and run
